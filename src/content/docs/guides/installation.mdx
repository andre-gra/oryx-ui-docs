# Installation

Installing Oryx UI is straightforward. Follow these steps to get it up and running in your project.

## Prerequisites

Before you begin, ensure you have:
- Node.js (version 18 or higher)
- npm, yarn, or pnpm
- React 17+ installed in your project

## Step 1: Install Oryx UI

### ⚠️ Please note that the project is currently in alpha and is not ready for production use !!

You can install Oryx UI using your preferred package manager:

```bash
# Using npm
npm install @andre-gra/oryx-ui

# Using yarn
yarn add @andre-gra/oryx-ui

# Using pnpm
pnpm add @andre-gra/oryx-ui
```

## Step 2: Set up the Provider

Wrap your application with the `OryxProvider` to enable theming and the AI agent:

```tsx
import { OryxProvider } from '@andre-gra/oryx-ui';
import '@andre-gra/oryx-ui/styles.css';

function App() {
  return (
    <OryxProvider defaultTheme="theme-amber">
      <YourApp />
    </OryxProvider>
  );
}
```

### Provider Options

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `defaultTheme` | `string` | `"theme-amber"` | The initial theme to use |
| `enableAgent` | `boolean` | `true` | Enable the AI theme agent |

## Step 3: Import and Use Components

Once the provider is set up, you can start using Oryx UI components:

```tsx
import { Accordion, Select, AlertDialog } from '@andre-gra/oryx-ui';

function MyComponent() {
  return (
    <Accordion
      items={[
        { mainText: "Section 1", collapsibleText: "Content 1" },
        { mainText: "Section 2", collapsibleText: "Content 2" }
      ]}
    />
  );
}
```

## Peer Dependencies

Make sure you have the required peer dependencies installed:

```json
{
  "react": ">=17.0.0",
  "react-dom": ">=17.0.0"
}
```
