# Introduction

Welcome to the **Oryx UI** documentation!

Oryx UI is a React component library with intelligent theming based on **Radix UI primitives**. It features an AI-powered theme agent that learns your preferences and automatically applies optimal theme combinations.

## Available Components

Oryx UI provides a growing set of components built for modern web applications:

- **Select** - Dropdown list of options
- **AlertDialog** - Modal dialogs for important actions
- **NavigationMenu** - Responsive navigation menus
- **Popover** - Rich content in a portal
