# Input

The `Input` component is a basic form control that enables users to enter and edit text.

## Usage

```jsx
import { Input } from '@oryx-ui/core';
import { html } from 'htm/preact';

function MyInputExample() {
  return html`
    <${Input} placeholder="Enter your name" />
    <${Input} value="Pre-filled value" />
    <${Input} type="password" placeholder="Enter password" />
    <${Input} disabled placeholder="Disabled input" />
  `;
}
```

## Props

| Prop        | Type      | Default   | Description                          |
| :---------- | :-------- | :-------- | :----------------------------------- |
| `value`     | `string`  | `''`      | The current value of the input.      |
| `onChange`  | `Function`|           | Callback fired when the input value changes. |
| `placeholder`| `string` |           | Text to display when the input is empty. |
| `type`      | `string`  | `'text'`  | The type of the input (e.g., `text`, `password`, `email`). |
| `disabled`  | `boolean` | `false`   | If `true`, the input will be disabled. |

## Examples

