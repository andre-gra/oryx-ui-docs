# NavigationMenu

The `NavigationMenu` component provides a responsive and accessible navigation system with support for dropdown menus, cards, and simple links.

## Import

```tsx
import { NavigationMenu } from 'oryx-ui';
```

## Usage

```tsx
import { NavigationMenu } from 'oryx-ui';

<NavigationMenu
  items={[
    {
      title: "Products",
      item: [
        { type: "card", title: "Featured", href: "/", text: "Our main product" },
        { type: "text", title: "Other", href: "/other", text: "Description" }
      ]
    },
    { title: "About", href: "/about" }
  ]}
/>
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `items` | `NavItem[]` | Required | Array of navigation items |

### NavItem

Navigation items can be either a dropdown menu or a simple link:

| Property | Type | Description |
|----------|------|-------------|
| `title` | `string` | Display text for the navigation item |
| `href` | `string` | (Optional) Direct link URL - makes it a simple link |
| `item` | `NavSubItem[]` | (Optional) Sub-items for dropdown - makes it a dropdown menu |

### NavSubItem

| Property | Type | Description |
|----------|------|-------------|
| `type` | `"card" \| "text"` | Display style of the sub-item |
| `title` | `string` | Title of the sub-item |
| `href` | `string` | Link URL |
| `text` | `string` | Description text |

## Accessibility

The NavigationMenu includes:

- Full keyboard navigation
- ARIA menu pattern implementation
- Focus management for dropdowns
- Escape key closes menus
